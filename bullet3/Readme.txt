1.  Add a bullet source tree. Prefered source is https://github.com/bulletphysics/bullet3. 
    Download and unpack a zip archive or repo and name the result after the following scheme:
    bullet-<version>-src. <version> might be a number or arbitrary string.
2.  Set the required version in config.bat (Package_Version) accordingly.
3.  Check the settings of ..\common\config.bat and fire up build, install or package depending on the task.
